组件之间的传值

1. 父子组件间的传值
1. 父组件给子组件传值

   在父组件调用子组件时，给子组件绑定自定义的属性。

   <List :city="selectCity"/>

   在子组件中使用props进行接收。在子组件中当成属性使用。

   props: ['city']

   props属于外部属性，只可以读取，不可以设置(单向数据流)
   props在接受外部属性时，可以设置为数组形式，也可以设置为对象的形式。

2. 子组件给父组件传值

    方式1：(建议使用以方便区分传值的方向)
    在父组件调用子组件时，给子组件绑定自定义事件。
      <Add @add="handleAddAction"/>
    在子组件内部，需要给父组件传值时，触发该自定义事件。
      this.$emit('add', this.text);

    方式2：
    在父组件调用子组件时，给子组件绑定自定义的属性，但是自定义属性值是函数。
      <Add :add="handleAddAction"/>
    在子组件内部，使用props接收属性，需要给父组件传值时，调用属性对应的函数，进行传值。
        props: {
            add: Function
        },
      this.add(this.text);

5. slot插槽的使用
当外部组件需要向内部组件插入某些数据或者另外的一个组件时就需要使用slot组件

一个slot组件会将所有的插入数据插入到slot组件创建的地方

如果需要制定插入特定的数据就需要给slot一个name属性，并在外部用name来向指定的插槽插入数据

6. ref的使用
ref 给组件中的标签设置的属性
设置属性需要在组件的dom结构中是唯一的
在组件的js实现中，可以通过this.$refs访问到对应的组件或dom结构

ref给普通标签设置得到的是dom元素
给组件标签设置得到的是组件对象

ref设置在了v-for指令作用的标签上，通过this.$refs得到就是一个数组，数组中是遍历出来的所有对象
